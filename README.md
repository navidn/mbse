# MBSE Handbook Using MATLAB and System Composer

This repository contains an educational handbook focused on the application of Model-Based Systems Engineering (MBSE) using MATLAB and System Composer. Developed as a practical teaching and learning resource, this handbook supports instruction in systems modeling, architecture definition, and analysis for engineering education.

## Contents

This handbook introduces key concepts, tools, and methodologies in Systems Engineering (SE) and Model-Based Systems Engineering (MBSE), with a strong emphasis on implementation using MATLAB and System Composer. It guides learners through the process of building, simulating, and validating system models using a structured, hands-on approach. By the end, readers will be prepared to model and analyze engineered systems using model-based methods.

### Modules

The handbook is divided into five instructional modules, each building on the previous one to create a cohesive learning path. These modules are supported by two practical case studies: the Fly-by-Wire Flight Control System and an Online Ordering System.

**Module 1: Introduction to Systems and Systems Engineering**  
Covers the definition of systems, systems thinking, behavior and structure, the system life cycle, and compares document-based and model-based systems engineering approaches.

**Module 2: System Requirements and Problem Definition**  
Focuses on defining stakeholder needs, articulating system-level problems, developing system requirements, and establishing the system context and boundaries.

**Module 3: Functional Analysis and System Behavior**  
Introduces function decomposition, functional hierarchies, flow block diagrams (FFBD), sequence and activity diagrams, and traceability matrices like N² and function-requirement maps.

**Module 4: Functional Allocation and System Components**  
Covers the allocation of functions to components, design of component hierarchies and flow, use of component-level diagrams and traceability tools (e.g., function-component and component-requirement matrices).

**Module 5: System Simulation and Model Validation**  
Explains how to verify and validate models using MATLAB System Composer and Simulink Test, including simulation of the Fly-by-Wire system. This module emphasizes closing the loop between requirements and system behavior.

### Case Studies and Application

Realistic system examples are used throughout, ranging from the Fly-by-Wire Flight Control System to an Online Ordering System. These examples illustrate the process from system definition to simulation and validation, showing how MBSE tools support both conceptual design and practical engineering analysis.
